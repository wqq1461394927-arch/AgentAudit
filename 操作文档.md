# 模块5：Settlement & Reputation 操作文档

## 模块定位

**模块5：结算与信誉系统** - 独立完整的漏洞奖金结算与信誉管理模块

负责：
- 项目方 Accept/Challenge 漏洞
- 7天超时自动接受
- 奖励自动分配（70%/20%/10%）
- Agent 校准分数管理
- 审计师信誉系统

- Node.js 20+
- PostgreSQL (或使用 SQLite)
- Foundry (可选，用于合约测试)

### 步骤

```bash
cd e:\模块5\backend

# 1. 安装依赖
npm install

# 2. 配置 .env
# 将 .env 文件中的 DATABASE_URL 修改为你的数据库地址

# 3. 初始化数据库
npx prisma migrate dev --name init

# 4. 启动服务
npm start
```

服务将在 `http://localhost:3005` 启动

## API 接口文档

### 1. 结算接口 (Settlement)

#### 创建漏洞
```
POST /api/v1/settlement/vulnerabilities

Body:
{
  "vulId": "0xvul001",
  "title": "Reentrancy in withdraw()",
  "description": "Critical vulnerability in withdraw function",
  "bounty": 1000,
  "submitter": "0x1234...",
  "projectAddress": "0x5678...",
  "projectName": "DeFi Protocol"
}
```

#### 接受漏洞
```
POST /api/v1/settlement/vulnerabilities/:vulId/accept
```

#### 挑战漏洞
```
POST /api/v1/settlement/vulnerabilities/:vulId/challenge

Body:
{
  "reason": "Not a real vulnerability"
}
```

#### 添加提交
```
POST /api/v1/settlement/vulnerabilities/:vulId/submissions

Body:
{
  "subVulId": "0xsub001",
  "submitter": "0x1234...",
  "agentId": "agent-001",
  "confidence": 95,
  "qualityScore": 85,
  "ranking": 1
}
```

#### 结算奖励
```
POST /api/v1/settlement/vulnerabilities/:vulId/settle

Body:
{
  "calibrationMultipliers": [120, 110, 100]
}
```

#### 漏洞列表
```
GET /api/v1/settlement/vulnerabilities
```

### 2. 声誉接口 (Reputation)

#### 注册审计师
```
POST /api/v1/reputation/auditors

Body:
{
  "wallet": "0x1234...",
  "name": "Alice"
}
```

#### 更新声誉
```
POST /api/v1/reputation/auditors/:id/update

Body:
{
  "newSubmission": true,
  "valid": true,
  "reward": 500
}
```

#### 审计师排行榜
```
GET /api/v1/reputation/auditors/leaderboard?limit=10
```

---

### 3. 校准接口 (Calibration)

#### 注册 Agent
```
POST /api/v1/calibration/agents

Body:
{
  "agentId": "agent-001",
  "name": "VulnHunter"
}
```

#### 上报预测结果
```
POST /api/v1/calibration/agents/:id/report

Body:
{
  "confidence": 95,
  "actualValid": true
}
```

#### 获取校准乘数
```
GET /api/v1/calibration/agents/:id/multiplier
```

#### Agent 排行榜
```
GET /api/v1/calibration/agents/leaderboard?limit=10
```

## 核心功能说明

### 1. 漏洞管理

- 创建漏洞 → 接受/挑战 → 超时自动接受（7天）

### 2. 挑战保证金

- 保证金 = MAX(20% 奖金, 50 USDC)
- 项目方输 → 保证金归发现者
- 项目方赢 → 保证金退还

### 3. 奖励分配

- 第一名：70%
- 第二名：20%
- 第三名：10%

### 4. AI 校准

- 记录 Agent 预测置信度与实际结果
- 计算校准分数 = 正确预测的置信度平均值
- 应用乘数：
  - 校准 ≥ 90% → 1.2x
  - 校准 ≥ 70% → 1.1x
  - 校准 ≥ 50% → 1.0x
  - 校准 < 50% → 0.5x

### 5. 声誉系统

审计师等级：
- Rookie (0-100)
- Bronze (100-500)
- Silver (500-2000)
- Gold (2000-5000)
- Elite (>5000)

## 〇、启动说明

1. 确保所有终端已关闭（无 node 残留）
2. 双击项目根目录 `start.bat`
3. 脚本自动：清理旧进程 → 并行启动 3 个服务 → 等待 8 秒 → 验证端口 → 打开浏览器
4. 终端显示端口验证结果（绿色 OK / 红色 NOT READY）
5. 浏览器自动打开 `http://localhost:5173`。如页面空白，F5 刷新即可

三服务架构：
  :3005  Module5 API  — SQLite 数据库 + REST API（漏洞结算/声誉/校准）
  :3000  Gateway API  — 统一网关（任务/Agent/赏金接口，降级 mock）
  :5173  Frontend     — React SPA，5 模块共享 store，sessionStorage 持久化

数据库重置：POST http://localhost:3005/api/reset （恢复工厂演示数据）
